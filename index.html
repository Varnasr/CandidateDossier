Why this matters

District teams need clean, comparable dossiers. We separate data (CSV) from layout (LaTeX) so every dossier prints consistently and is easy to review and share.

Privacy (mandatory): All phone numbers are automatically masked to ******1234 before any processing or PDF generation. This behaviour cannot be disabled. Upload only what’s necessary. Use for non-partisan, administrative documentation under the Indian IT Act. Outputs are provided “AS IS”.

How it works

  1. Prepare your CSV with the required headers (see Easy tab).
  2. Go to Easy → Upload CSV → Get PDF.
  3. If serverless compile is unavailable, use Backup with Overleaf.

Backup (Advanced) — Overleaf

  1. Use the same CSV; we convert to masked LaTeX blocks locally.
  2. Download candidate_blocks.tex and main.tex.
  3. Open Overleaf → New Project → upload both files.
  4. Ensure \input{candidate_blocks.tex} exists in main.tex.
  5. Recompile → Download PDF.
